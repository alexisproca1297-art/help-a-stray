# Help A Stray: Animal Adoption Web App

Help A Stray is a web-based animal rescue and adoption management system developed using HTML, CSS, PHP and MySQL.

The system aims to support animal rescue centres by allowing administrators to manage animal profiles and adoption applications, while users can view animals available for adoption and submit adoption applications online.

## Technologies

- HTML
- CSS
- PHP
- MySQL
- phpMyAdmin
- cPanel

## Architecture

The project will follow a simple three-tier structure:

- Presentation layer: public pages and admin interface
- Application layer: controllers, models and helper functions
- Data layer: MySQL database and database connection files

## Planned Core Features

- Animal listing page
- Animal profile pages
- Adoption application form
- Administrator login
- Animal management
- Adoption application management
