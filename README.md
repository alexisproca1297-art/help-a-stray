# Help A Stray: Animal Adoption Web App

Help A Stray is a web-based animal rescue and adoption management system developed as a Computing Level 6 project. The application is designed to support animal rescue centres by allowing animal profiles and adoption applications to be managed through a simple web-based system.

## Project Aim

The aim of this project is to design, develop and test a functional prototype of a web-based animal rescue and adoption management system using HTML, CSS, PHP and MySQL.

## Technologies Used

- HTML
- CSS
- PHP
- MySQL
- phpMyAdmin
- XAMPP
- GitHub Desktop

## Core Features

### Public User Area

- View animals available for adoption
- Search animals by name
- Filter animals by species and status
- View animal profile details
- Submit an adoption application form

### Administrator Area

- Admin login
- Admin dashboard with statistics
- View recent adoption applications
- Add animal profiles
- Edit animal profiles
- Delete animal profiles
- View adoption applications
- Approve or reject adoption applications
- Upload and display animal images

## Database Tables

- `admins`
- `animals`
- `applications`

## Architecture

The project follows a simple three-tier structure:

- Presentation Layer: public pages and administrator interface
- Application Logic Layer: PHP scripts, validation and session handling
- Data Layer: MySQL database and database connection
