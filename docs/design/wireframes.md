# Help A Stray Wireframe Notes

## Home Page

The home page includes a main welcome section, a call-to-action button, system statistics, featured animal cards and a short project description.

## Animals Page

The animals page includes a search bar, species filter, status filter and a responsive grid of animal cards. Each card displays the animal image, name, species, breed, status and a link to the animal details page.

## Animal Details Page

The animal details page displays the selected animal image, name, species, breed, age, gender, adoption status and description. If the animal is available, the user can access the adoption application form.

## Adoption Application Page

The adoption application page displays the selected animal information and a form where users can enter their name, email, phone number, address, housing type and previous pet experience.

## Admin Dashboard

The admin dashboard includes system statistics such as total animals, available animals, adopted animals and application counts. It also includes recent applications and navigation links to management pages.

## Manage Animals Page

The manage animals page displays all animal records in a table. Administrators can add, edit or delete animal profiles.

## Applications Page

The applications page displays submitted adoption applications in a table. Administrators can view full application details and update the application status.
