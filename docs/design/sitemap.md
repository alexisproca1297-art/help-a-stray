# Help A Stray Sitemap

## Public User Area

- Home
  - View system overview
  - View featured animals
  - Navigate to available animals

- Animals
  - View all animals
  - Search by name
  - Filter by species
  - Filter by adoption status

- Animal Details
  - View full animal information
  - View animal image
  - Apply to adopt if available

- Adoption Application
  - Complete adoption form
  - Submit application

- About
  - View project purpose
  - View system objectives
  - View technologies used

- Contact
  - View rescue centre contact information
  - View opening hours
  - Use contact form prototype

## Administrator Area

- Admin Login
  - Authenticate administrator

- Admin Dashboard
  - View system statistics
  - View recent applications
  - Navigate to management pages

- Manage Animals
  - View all animals
  - Add animal
  - Edit animal
  - Delete animal

- Adoption Applications
  - View submitted applications
  - View application details
  - Update application status
